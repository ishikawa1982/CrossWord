# 変更履歴

このファイルは CrossWord のバージョンごとの変更点をまとめたものです。
バージョンは [セマンティックバージョニング](https://semver.org/lang/ja/) に従い、新しい順に記載します。
（バージョンの運用ルールは `CLAUDE.md` を参照）

## [0.15.0] - 2026-06-15
- 対戦のリザルト画面に、そのゲームで出題された問題と答えの一覧を順位欄の下に表示するようにした
- ゲーム終了時はサーバが解答付きの盤面を配信するようにし、答えを表示できるようにした
- 新ジャンル「人体」を追加（臓器・骨・筋肉・体の部位など56問）
- 新ジャンル「スポーツ」を追加（競技・用具・ルール用語など60問）

## [0.14.1] - 2026-06-15
- ルームコード入力で濁点・半濁点のひらがな（び・ぐ・ば 等）や「わ」が打てず、`へび`・`まぐ`・`わに` 等を含むコードに参加できなかった不具合を修正
- 入力フィルタを固定文字リストからひらがな全体（U+3041〜U+3096）の許可に変更
- 未使用だったサーバ側の `CODE_CHARS` 定数を削除

## [0.14.0] - 2026-06-14
- 全ジャンルの問題を大幅に増量し、各ジャンルをバランスよく拡充（日本語計2941問）
- 各ジャンルの問題数: 昭和309 / グルメ299 / 教養300 / 動物・植物300 / エンタメ292 / 令和289 / サブカル254 / 音楽298 / 地理・世界300 / 歴史300
- 追加分はカナのみ・既存と重複しないよう自動検証して投入

## [0.13.0] - 2026-06-14
- 新ジャンル「地理・世界」を追加（国・首都・都市・名所・山や川などをベースに179問）
- 新ジャンル「歴史」を追加（歴史上の人物・出来事・時代・遺跡などをベースに163問）
- 日本語の問題数を計2274問に
