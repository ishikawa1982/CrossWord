# CrossWord — 対戦型クロスワードパズル

最大4名でネットワーク対戦できるクロスワードゲーム。カギに早押しで正解すると、
その単語のマスが自分の色に塗られます。**最新の正解者で上書き**されるため、
交差するマスを別の単語で奪い合えます。全マスが埋まった時点で、自分の色のマスが
最も多いプレイヤーの勝ちです。

## 特徴
- 最大4人のリアルタイム対戦（Socket.IO）
- 単語単位・正解必須。誤答すると数秒のクールダウン
- 盤面は単語＋カギのリストから**自動生成**
- 日本語（かな）・英語の両対応
- ルームコードで合流（ログイン不要）

## 技術構成（モノレポ / npm workspaces）
- `shared/` — 型定義・クロスワード生成・ゲームロジック（クライアント/サーバ共有, TypeScript）
- `server/` — Node.js + Express + Socket.IO（サーバ権威。解答はサーバ内のみ保持）
- `client/` — React + TypeScript + Vite

## セットアップ
```bash
npm install
```

## 開発（サーバ＋クライアント同時起動）
```bash
npm run dev
```
- クライアント: http://localhost:5173
- サーバ:       http://localhost:3001

ブラウザのタブを複数開き、1つで「部屋を作成」→表示されたルームコードを他タブで入力して
「参加」→ホストが「ゲーム開始」で対戦できます。

## テスト
```bash
npm test          # shared のゲームロジック・生成器の単体テスト（vitest）
```

## ビルド / 本番起動
```bash
npm run build     # 全ワークスペースの型チェック＋クライアントの本番ビルド
npm start         # サーバ起動（既定 PORT=3001、環境変数 PORT で変更可）
```
クライアントのサーバ接続先は環境変数 `VITE_SERVER_URL` で変更できます（既定 `http://localhost:3001`）。

## ゲームルール詳細
- カギ（ヨコ/タテ）を選び、答えを入力して回答。正解ならその単語の全マスが自分の色に。
- すでに他人が取得済みの交差マスも、正解すれば上書きして奪える。
- 誤答すると約5秒のクールダウン（総当たり防止）。
- 全マスに所有者が付いたらゲーム終了。色の多い人が勝ち（同点は引き分け）。

## 単語データの追加
`server/src/data/words.ja.json` / `words.en.json` に `{ "answer", "clue" }` を追加すると
出題プールが増えます。生成器がランダムにサブセットを選んで盤面を構築します。
